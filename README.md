# USA Auto Hub Mapper

米国内の自動車関連製造拠点を可視化・管理するインタラクティブなマップアプリケーション。

## 機能

### マップビュー
- **ホバーツールチップ**: 拠点ポイントにマウスを近づけると、詳細情報が表示されます
  - メーカー名
  - 所在地（都市、州）
  - 生産品目
  - 生産能力
  - 詳細情報
  - カテゴリーバッジ
- **クリックポップアップ**: マーカーをクリックすると、より詳細な情報がポップアップで表示されます

### フィルタリング機能
- **カテゴリーフィルター**: 日系メーカー、米国メーカー、欧州メーカー、EVメーカー、バッテリーメーカー
- **工程フィルター**: 組立、バッテリー製造、部品製造、パワートレイン
- **検索**: メーカー名、州、都市、生産品目で検索

### テーブルビュー
- ソート機能付きのデータテーブル
- 行をクリックしてマップ上の拠点にジャンプ

### 統計ダッシュボード
- 総拠点数
- 日系メーカー数
- EV/バッテリー系拠点数
- 主要生産州

### エクスポート
- CSV形式でデータをエクスポート可能

## 技術スタック

- **Leaflet**: 地図表示
- **Tailwind CSS**: スタイリング
- **Lucide Icons**: アイコン

## データ

15拠点の製造拠点データを含んでいます：
- トヨタ、ホンダ、日産（日系メーカー）
- GM、フォード（米国メーカー）
- テスラ、リビアン（EVメーカー）
- BMW、ボルボ（欧州メーカー）
- LGエナジーソリューション、パナソニック（バッテリーメーカー）
